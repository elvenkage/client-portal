# Role System

The system has two main user groups.

Internal Users (Agency Team)

super_admin
admin
project_manager
team_member

External Users (Client Portal)

client

Role descriptions

super_admin
Full system access.

admin
Administrative access excluding system ownership.

project_manager
Manages projects, tasks, team members.

team_member
Works on assigned tasks.

client
External user with limited access to their own projects.
