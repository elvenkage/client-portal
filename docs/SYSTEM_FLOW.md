# System Flow

1. Agency creates internal users.

2. Agency creates projects.

3. Project manager assigns team members to the project.

4. Project manager invites client users.

5. Client users are added to the project via project_members.

6. Milestones are created.

7. Tasks are created under milestones.

8. Team members work on tasks.

9. Files and comments are attached to tasks.

10. Activity logs record important actions.

11. Clients can log in and monitor project progress.
